Install R Studio: https://download1.rstudio.org/desktop/windows/RStudio-1.3.1093.exe
1)open swarn.rmd file in R studio
2)in top right corner there is an option called "install", Now install  packages  flexdashboard, DT,Tidyverse ( it would ask to install required packages by default)
3)press ctrl+shift+k to open the dashboard created
4)It will ask to select Transaction-sheet1 (which is our required dataset)
5)Dashboard will run successfully
